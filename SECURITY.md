# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in NotTodayHoney, please report it through [GitHub Security Advisories](https://github.com/Vinksyunit/NotTodayHoney/security/advisories/new).

**Do not open a public issue for security vulnerabilities.**

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

Only the latest minor release receives security fixes.

## Disclosure Policy

- We will acknowledge your report within **48 hours**
- We aim to release a fix within **7 days** for critical issues
- We will coordinate the disclosure timeline with you
